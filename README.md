# 2Dit Jr. Frontend Test

## Technology stack

- HTML 5
- Dart SASS
- PostCSS
- JavaScript ES6
- Webpack 5

## Инструкция по сборке проекта

1. Склонировать репозиторий
2. Выполнить команду `npm i` для установки зависимостей
3. Выполнить одну из следующих команд:

### Production Build

```
npm run build
```

### Development Build

```
npm run build-dev
```

### Development Server

```
npm start
```
